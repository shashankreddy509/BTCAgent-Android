4 Development NEXT

Two parts: one-time foundation, then a repeatable loop per screen.
Phase 0 — Foundation (no TDD loop, just wiring that compiles)
1Build wiring: Java 11→17 + add Hilt/KSP/Retrofit/Firebase/Vico to catalog
2INTERNET permission + @HiltAndroidApp App + manifest
3Empty di/ core/ data/ ui/ skeleton + 4 Hilt modules
4google-services.json + verify assembleDebug green
Exit criteria: app builds with the full DI graph + can mint a Firebase token.
Per-screen loop — the 9 inner dev steps (repeated per screen)
1Branch
2Contract (DTO+API+repo sig)
3Write failing tests (TDD)
4Implement repo→VM→screen
5Wire Hilt DI
6Run tests → green
7Manual verify on device
8Code review
9Commit + changelog
Maps to agent pipeline planner → test-writer → implementer → test-runner → code-review-coordinator → changelog.
Build order: Login → Gate → Dashboard (proves WS+auth) → read-only → control → admin.
